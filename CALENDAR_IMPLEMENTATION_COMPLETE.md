## üìä Data Integration

### API Endpoint Used
- **`/api/automated-signals/dashboard-data`** (GET)
  - Returns: `active_trades`, `completed_trades`, `stats`, `hourly_distribution`, `session_breakdown`
  - Combines active and completed trades for calendar display

### Data Processing
```javascript
signals.forEach(signal => {
    if (signal.date) {
        tradesByDate[dateStr]++;
        
        // Track active trades
        if (signal.trade_status === 'ACTIVE' || signal.trade_status === 'CONFIRMED') {
            activeByDate[dateStr]++;
        }
        
        // Calculate average MFE
        const mfe = signal.final_mfe || signal.current_mfe || 0;
        statsByDate[dateStr].totalMFE += mfe;
        statsByDate[dateStr].count++;
    }
});
```
